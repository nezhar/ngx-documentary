What is this?

ngx-documentary Is a package that allows you do quickly create a Demo/Documentation page for an Angular package, without having to worry about layout, routing etc.

The module contains a single component that you have to ann in the Angular bootstrap component. You only have to create view components that will be loaded using the DocumentaryConfig.
